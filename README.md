## 기능

* 주요 브랜드 카페 상품 표시
  * 카테고리 필터 (커피, 차, 블렌디드, 스무디, 주스, 에이드, 그 외)
  * 상품 검색 기능
  * 매일 상품 정보 자동 업데이트
* 사용자 관리
  * 구글, 카카오, 네이버 로그인
  * 사용자 이름, 핸들 수정
  * 작성한 후기 및 점수 통계 표시
* 상품 후기
  * 상품 페이지에서 후기 작성 가능
    * 점수는 다음 단계 중에서 선택. 최악(1), 별로(2), 보통(3), 좋음(3.5) 추천(4), 강력 추천(4.5), 최고(5)
    * 후기 최대 글자 수 500자.
    * 최대 사진 2장 추가 가능
  * 상품 페이지에서 후기 통계 표시
    * 점수 히스토그램, 평균 점수, 후기 개수
